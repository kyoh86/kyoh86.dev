スイッチ向け: WACUL マインクラフト サーバーに入るまでのガイドライン

ウインドウズ環境でプレイしたい方は ウインドウズ環境向けガイドライン へ。

サーバー接続までの手順​

  • マインクラフト（統合版）の購入
  • Xbox ゲーマーアカウントの用意
  • マインクラフトで Xbox ゲーマーアカウントにログイン
  • サーバーへの接続設定

手順ごとに説明します。

マインクラフトの購入​

スイッチ用のソフトを購入してください https://www.minecraft.net/ja-jp/store/minecraft-switch

Xbox ゲーマー アカウントの用意​

Xbox ゲーマーアカウントは、スマホやパソコンのブラウザで Xbox live から作ることができます。

  1. すでに持っている可能性があるので、サインインを試してみましょう。
  2. 持っていない場合はアカウントを作成しましょう。
  3. アカウントが用意できたら、 Xbox プロファイル ページにアクセスしてみましょう。ログインできていれば、自分のアカウント名が表示されます。

マインクラフトで Xbox ゲーマーアカウントにログイン​

  1. スイッチでマインクラフトを起動します。
  2. オープニング画面の左下の「ニンテンドーアカウントでログイン」で、ニンテンドーのアカウントでログインします。
    • (画像割愛)
  3. 左下の「マイクロソフトアカウントでログイン」が表示されるので、マイクロソフトのアカウントでログインします。
  4. ダイアログが表示されるので、その案内に従って、スマホやパソコンで表示されたURLにアクセスします。
    • コードの入力を求められるので、表示されたコードを入力しましょう。
  5. 確認のため、スマホやパソコンのブラウザで Xbox プロファイル ページにアクセスしてみましょう。
    - 大きく表示されるのが Xboxゲーマータグ（アカウント名）です。<img src="https://i.gyazo.com/af7dfdbc8b0899267a1121cc5956539b.png" width="592" height="197" />
    - 正常にログインできていれば、マインクラフトの左下にがXboxゲーマータグが表示されます。

アクセス許可をもらう​

Xboxゲーマータグをサーバー管理者（山田）まで教えて下さい。

Xboxゲーマータグがわからない方は、スマホやパソコンのブラウザで Xbox プロファイル ページにアクセスしてみましょう。大きく表示されるのが Xboxゲーマータグ（アカウント名）です。

  • 管理者側で接続許可リストに追加する作業をします。
  • サーバーの再起動が必要なため、誰かがゲームに参加している場合は、翌日などまで待ってもらうかもしれません。

サーバーに接続する​

サーバーに接続するには、

  1. スイッチ本体の設定変更
  2. マインクラフトで接続先を変える

の2ステップが必要です。

スイッチ本体の設定変更​

スイッチは「ライフボート」といった公式特集サーバーにしか入れないようになっています。WACULマインクラフトサーバーに参加するには、スイッチ本体の設定変更が必要です。

注意点​

  • ※1. この設定は使用しているインターネット環境が変わるたびに設定する必要があります。別のインターネット環境（Wi-Fi環境）で遊ぶときは、またこの設定を行う必要があります
  • ※2. 必ず大人の許可・監督の下で行ってください。

  1. スイッチ本体の「設定」を開きます。

  2. 「インターネット」の一番上にある「インターネット設定」を開きます。

  3. 接続できるネットワークの一覧が出てきます。チェックが付いているネットワークを選んで、「設定の変更」に進みます。

  4. 「DNS設定」という項目を、以下のように変更します。

    • DNS設定: 手動
    • 優先DNS: 104.238.130.180
    • 代替DNS: 008.008.008.008

終わったら「保存する」を押してください。

この番号はどこから取ってきたのか​

とつぜん12桁の番号を入力させられて、怪しいと思ったことでしょう。この数字はどこから出てきたのでしょうか。 実は、アメリカのMatthewさんが、こちらのコードを公開しています。 彼は、ゲーム機で特集サーバー以外が遊べるように、いろいろな人と協力して、特殊な "DNSサーバー" をネット上に公開しています。 そのIPアドレスが上記の数字です。

マインクラフトでWACULサーバーに接続する​

マインクラフトの起動画面から、右上の「サーバー」というタブを開き、特集サーバーの どれでもいいので、適当に 参加してください。すると、 Server List という画面が表示されます。

Server List という画面が表示されず、いきなりゲーム画面に入ってしまった場合は、スイッチ本体の設定変更に失敗しているかもしれません。もう一度設定を確認してください。

Connect to a Server を選んでください。

サーバーの接続先を設定する画面が開くので、以下の通り入力してください。

  • Server Address: サーバー管理者（山田）にお問い合わせください。お知らせします。
  • Server Port: 19132
  • Display Name on Server List: WACUL（わかりやすい名前なら何でもいい）
  • Add to Server List: オン（右側にトグルして緑色になっている）

もとの Server List という画面に戻るので、登録した名前（Display Name）のボタンで接続できます。
